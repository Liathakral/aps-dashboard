# APS Security Dashboard

A responsive cybersecurity scanning dashboard built using **Next.js, React, and TailwindCSS**.

## Features

- Authentication page (Login / Signup)
- Dashboard with scan overview
- Active Scan monitoring UI
- Live scan console
- Finding log
- Responsive design (mobile, tablet, desktop)
- Dark / light theme support
- Interactive UI components

## Tech Stack

- Next.js (App Router)
- React
- TailwindCSS
- Lucide Icons
- React Icons
- Sonner (toast notifications)
